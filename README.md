# 東京電子専門学校 卒業・進級制作展示会2022

東京電子専門学校の情報システム系の中に、ウェブ・メディア科があり、こちらでは毎年、年度内最後のイベントとして学生生活の集大成となる作品を制作し展示する、卒業・進級展示会を開催いたしております。

https://2022graduation.tokyo-ec.net/
